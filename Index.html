Este é um gerador de senhas moderno e responsivo, desenvolvido com HTML, CSS e JavaScript. Ele permite ao usuário criar senhas seguras com opções personalizáveis, como: ✅ Escolher o tamanho da senha ✅ Incluir letras maiúsculas ✅ Incluir números ✅ Incluir símbolos ✅ Botão para gerar nova senha ✅ Botão para copiar a senha O design pode ser personalizado com diferentes cores e imagens.
